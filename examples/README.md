# LtL Course Builder — Examples

This folder contains example training projects created using the LtL Course Builder toolkit.

(Examples will be added as sample projects are completed and approved for sharing.)

## Structure

Each example should include:
- `overview-session/brainstorming-notes.md`
- `overview-session/overview.md`
- `training-modules/<slug>/<slug>.md` files for each module
- `training-materials/` folder with generated outputs

---

## Contributing an Example

To share your training design as an example:

1. Ensure all modules are complete and outputs generated
2. Remove any confidential participant information
3. Create a subfolder: `examples/<training-name>/`
4. Copy your project structure into it
5. Add a README explaining the training context and goals
6. Submit a PR to this repo

---

## Example Template

When adding an example, include a README.md in the example folder:

```markdown
# [Training Name]

**Purpose**: [One-sentence description]

**Duration**: [e.g., 1 day, 3 half-days]

**Participants**: [Target audience]

**Number of Modules**: [X]

**Key Features**: 
- [Notable design choice or innovation]
- ...

## How to Use This Example

1. Copy the entire folder to your projects
2. Adapt the module content to your context
3. Run `/ltl-generate all` to produce final materials
```
